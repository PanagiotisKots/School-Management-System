## Made for Windows platforms in C++

This project is a command-line interface (CLI) application designed to manage various aspects of a school system, 
including student records, course management, and staff information. The application is built using C++ and is specifically 
tailored for Windows platforms.

## 📚 Project Overview

This CLI system was developed as part of Educational Institute assignment for a C++ course. The goal of this project is to create 
a functional and user-friendly application that facilitates the management of a school's data through a terminal interface.

## 🤖 Features

Student Management: Add, view, update, and delete student records.
Course Management: Manage courses, including adding new courses and enrolling students.
Staff Management: Handle staff records and their assignments.
Search and Filter: Easily search and filter through student, course, and staff data.
Data Persistence: Save and load data from files to ensure that changes are preserved between sessions.

## 🛠️ Installation

To get started with the School Management CLI System, follow these steps:

 1. Clone the Repository:

        git clone https://github.com/PanagiotisKots/School-Management-System.git

2. Navigate to the Project Directory:

       cd School-Management-System

3. Compile the Code:

       Using g++ or a Development enviroment with integrated compiler for c/c++ (I use Dev C/C++ ide)

4. Run the Application:

       school_management.exe




## 👨‍💻 Usage

After running the application, you will be presented with a command-line interface where you can choose from 
various options to manage student, course, and staff information. Follow the on-screen prompts to perform the 
desired operations.
